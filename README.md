# Insurance_company_arch
Проект информационной системы для автоматизации страховой компании. Включает функциональные и объектные модели бизнес-процессов (IDEF0, IDEF3, DFD), логическую и физическую модели данных (IDEF1X), а также диаграммы прецедентов (UML) и расчет метрик программного обеспечения.
